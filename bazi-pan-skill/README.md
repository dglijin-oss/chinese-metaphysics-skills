# 八字排盘技能

> "易与天地准，故能弥纶天地之道" ——《周易·系辞》

八字命理，又称四柱命理，是以出生年月日时的天干地支推算命运的术数。本技能提供完整的八字排盘、十神计算、大运排法、格局判断、用神选取及流年推算功能。

## ✨ 核心功能

- ✨ 四柱排盘（年柱、月柱、日柱、时柱）
- ✨ 十神计算（比肩、劫财、食神、伤官等）
- ✨ 大运排法（8 步大运，每运 10 年）
- ✨ 五行统计（木火土金水数量及缺失检测）
- ✨ 格局判断（正官格、七杀格、食神格等 10 种格局）
- ✨ 用神选取（日主旺衰、喜用神、忌神）
- ✨ 流年推算（当年运势、冲克分析）
- ✨ 详细断语（事业、婚姻、财运、健康）
- ✨ 日主性格分析
- ✨ 纯 Python 实现，无外部依赖

## 🔧 技术特点

- 🔧 精确农历转换（1900-2100 年）
- 🔧 精确节气计算（天文算法）
- 🔧 真太阳时校正（支持城市/经度）
- 🔧 纯 Python 实现，无外部依赖
- 🔧 支持 JSON 格式输出
- 🔧 命令行界面友好

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/bazi-pan-skill.git
cd bazi-pan-skill

# 确保 Python 3.8+ 已安装
python3 --version

# 赋予执行权限
chmod +x scripts/bazi_pan.py
```

### 基本使用

```bash
# 基础排盘（当前时间）
python3 scripts/bazi_pan.py --date "1990-05-15" --hour 8

# 指定性别（影响大运顺逆）
python3 scripts/bazi_pan.py --date "1990-05-15" --hour 8 --gender 女

# 流年推算
python3 scripts/bazi_pan.py --date "1990-05-15" --hour 8 --liu-nian 2026

# 输出 JSON 格式
python3 scripts/bazi_pan.py --date "1990-05-15" --hour 8 --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --date | -d | 出生日期 (YYYY-MM-DD) | 必填 |
| --hour | -H | 出生时辰 (0-23) | 必填 |
| --gender | -g | 性别 (男/女) | 男 |
| --liu-nian | -l | 流年年份 | 可选 |
| --json | -j | 输出 JSON 格式 | 否 |
