## 1. Description

紫微斗数是中国传统命理学中最精密的星命术，以出生年月日时排布十四主星于十二宫位，推算一生命运。因其准确精细，被誉为"星命之宗"。

**核心功能 v2.0.0：**
- ✅ 十四主星排布（紫微/天机/太阳等）
- ✅ 十二宫位定位（命宫/兄弟/夫妻等）
- ✅ 四化飞星推算（化禄/化权/化科/化忌）
- ✅ 辅星杂曜排布（左辅/右弼/天魁/天钺等）
- ✅ 五行局计算（水二局/木三局等）
- ✅ 大限流年推算
- ✅ 命盘分析（性格/事业/婚姻/财运）

**v1.0.0 基础功能：**
- ✅ 十四主星基础排盘、十二宫位

## 3. How to use

### 调用方式

```bash
# 基础排盘
python3 ~/.openclaw/skills/ziwei-skill/scripts/ziwei_pan.py --date "1990-05-15" --hour 8

# 指定性别（影响大限顺逆）
python3 ~/.openclaw/skills/ziwei-skill/scripts/ziwei_pan.py --date "1990-05-15" --hour 8 --gender 女

# 指定流年
python3 ~/.openclaw/skills/ziwei-skill/scripts/ziwei_pan.py --date "1990-05-15" --hour 8 --liu-nian 2026

# 输出 JSON 格式
python3 ~/.openclaw/skills/ziwei-skill/scripts/ziwei_pan.py --date "1990-05-15" --hour 8 --json
```

## 8. 命盘分析规则

### 8.1 命宫分析

**主星特性：**
- 紫微/天府 → 领导型，适合管理
- 天机/太阴 → 谋略型，适合策划
- 太阳/天梁 → 公益型，适合服务
- 武曲/七杀 → 行动型，适合执行
- 天同/巨门 → 表达型，适合沟通
- 廉贞/贪狼 → 艺术型，适合创意
- 天相 → 辅佐型，适合幕僚
- 破军 → 开拓型，适合创业

### 8.2 四化分析

| 四化入宫 | 含义 |
|----------|------|
| 化禄入命 | 一生顺遂，有财禄 |
| 化权入命 | 有权力欲，事业心强 |
| 化科入命 | 有名声，利学业考试 |
| 化忌入命 | 多波折，需努力克服 |

### 8.3 大限流年

**大限：** 每 10 年一大限，看该 10 年运势
**流年：** 每年一流年，看当年运势

**推算规则：**
- 阳男阴女顺行，阴男阳女逆行
- 从命宫开始，依十二宫顺序
