# 风水堪舆技能

> "风水者，藏风得水也。" ——《葬书》

风水学是中国传统环境哲学，研究人与自然的和谐关系。八宅风水是风水学中最实用的流派之一，以住宅坐向和居住者命卦为基础，判断方位吉凶。本技能提供完整的八宅风水排盘和布局建议功能。

## ✨ 核心功能

- ✨ 八宅分类（东四宅/西四宅）
- ✨ 命卦计算（出生年份推算）
- ✨ 游年九星排布（伏位/生气/天医/延年/五鬼/六煞/祸害/绝命）
- ✨ 方位吉凶分析
- ✨ 布局建议（门窗/卧室/厨房/卫生间）
- ✨ 煞气识别与化解
- ✨ 纯 JavaScript 实现，无外部依赖

## 🔧 技术特点

- 🔧 精确命卦计算（1900-2100 年）
- 🔧 完整游年九星算法
- 🔧 方位吉凶判断
- 🔧 纯 JavaScript 实现，无外部依赖
- 🔧 支持 JSON 格式输出
- 🔧 命令行界面友好

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/fengshui-skill.git
cd fengshui-skill

# 确保 Node.js 14+ 已安装
node --version

# 赋予执行权限
chmod +x index.js
```

### 基本使用

```bash
# 基础排盘（按出生年份）
node index.js --year 1990

# 指定性别（影响命卦计算）
node index.js --year 1990 --gender 女

# 指定住宅坐向
node index.js --year 1990 --direction 坐北朝南

# 输出 JSON 格式
node index.js --year 1990 --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --year | -y | 出生年份 | 必填 |
| --gender | -g | 性别 (男/女) | 男 |
| --direction | -d | 住宅坐向 | 坐北朝南 |
| --json | -j | 输出 JSON 格式 | 否 |

## 📚 风水基础知识

### 八宅分类

| 宅型 | 卦象 | 方位 | 适合命卦 |
|------|------|------|----------|
| 东四宅 | 坎、离、震、巽 | 北、南、东、东南 | 东四命 |
| 西四宅 | 乾、坤、艮、兑 | 西北、西南、东北、西 | 西四命 |

### 命卦计算

**男性：**
```
(100 - 出生年份后两位) ÷ 9 取余数
```

**女性：**
```
(出生年份后两位 - 4) ÷ 9 取余数
```

余数对应卦象：
| 余数 | 卦象 | 五行 |
|------|------|------|
| 1 | 坎 | 水 |
| 2 | 坤 | 土 |
| 3 | 震 | 木 |
| 4 | 巽 | 木 |
| 5 | 男坤女艮 | 土 |
| 6 | 乾 | 金 |
| 7 | 兑 | 金 |
| 8 | 艮 | 土 |
| 9 | 离 | 火 |

### 游年九星

| 星名 | 吉凶 | 含义 |
|------|------|------|
| 生气 | 大吉 | 生机、活力、财运 |
| 天医 | 大吉 | 健康、医疗、贵人 |
| 延年 | 大吉 | 长寿、婚姻、稳定 |
| 伏位 | 小吉 | 平稳、保守、等待 |
| 五鬼 | 大凶 | 鬼怪、火灾、失窃 |
| 六煞 | 次凶 | 桃花、口舌、是非 |
| 祸害 | 次凶 | 疾病、意外、破财 |
| 绝命 | 大凶 | 绝症、死亡、大凶 |

### 布局原则

| 区域 | 宜设方位 | 忌设方位 |
|------|----------|----------|
| 大门 | 生气、延年、天医 | 五鬼、绝命 |
| 卧室 | 生气、天医、伏位 | 五鬼、绝命、六煞 |
| 厨房 | 天医、延年 | 西北方（火烧天门） |
| 卫生间 | 五鬼、绝命、祸害 | 生气、天医 |
| 书房 | 生气、文昌位 | 五鬼、绝命 |

### 常见煞气及化解

| 煞气 | 特征 | 化解方法 |
|------|------|----------|
| 五鬼煞 | 火灾、失窃 | 葫芦、五帝钱 |
| 六煞煞 | 桃花劫、口舌 | 水晶球、铜葫芦 |
| 祸害煞 | 疾病、意外 | 铜铃、八卦镜 |
| 绝命煞 | 大凶、重病 | 泰山石敢当、麒麟 |
| 穿堂煞 | 前后门直通 | 屏风、玄关 |
| 角煞 | 尖角对冲 | 凸面镜、植物 |

## 🐍 JavaScript API 调用

