# 择日学技能 v2.0.0

> "择吉避凶，顺天应时。" ——《协纪辨方书》

## 1. Description

择日学是中国传统选择吉时吉日的术数，用于婚嫁、开业、动土、出行等重要事项。通过查询黄历、神煞、建除等信息，选择最有利的时机。

**核心功能 v2.0.0：**
- ✅ 建除十二神查询（建/除/满/平等）
- ✅ 黄道黑道日查询
- ✅ 神煞吉凶查询（天德/月德/岁破等）
- ✅ 宜忌事项查询
- ✅ 冲煞查询（冲生肖/煞方位）
- ✅ 分类择日（婚嫁/开业/动土/出行等）
- ✅ 吉日推荐（月度/年度）

**v1.0.0 基础功能：**
- ✅ 建除十二神、基础宜忌

## 3. How to use

### 调用方式

```bash
# 查询指定日期吉凶
python3 ~/.openclaw/skills/ze-ri-skill/scripts/ze_ri.py --date "2026-04-15"

# 查询月度吉日
python3 ~/.openclaw/skills/ze-ri-skill/scripts/ze_ri.py --month "2026-04" --event "婚嫁"

# 查询年度吉日
python3 ~/.openclaw/skills/ze-ri-skill/scripts/ze_ri.py --year 2026 --event "开业"

# 指定生肖（避开冲日）
python3 ~/.openclaw/skills/ze-ri-skill/scripts/ze_ri.py --month "2026-04" --zodiac "鼠"

# 输出 JSON 格式
python3 ~/.openclaw/skills/ze-ri-skill/scripts/ze_ri.py --date "2026-04-15" --json
```

### 命令行参数

| 参数 | 简写 | 说明 |
|------|------|------|
| --date | -d | 查询日期 (YYYY-MM-DD) |
| --month | -m | 查询月份 (YYYY-MM) |
| --year | -y | 查询年份 |
| --event | -e | 事项类型（婚嫁/开业/动土/出行/搬家/签约/领证） |
| --zodiac | -z | 生肖（避开冲日） |
| --json | -j | 输出 JSON 格式 |

### 事项类型

| 类型 | 说明 |
|------|------|
| 婚嫁 | 结婚、订婚、领证 |
| 开业 | 开店、开张、开业典礼 |
| 动土 | 装修、建房、动土开工 |
| 出行 | 出差、旅游、远行 |
| 搬家 | 入宅、搬迁、移徙 |
| 签约 | 签合同、谈判、合作 |
| 求医 | 看病、手术、治疗 |
| 安葬 | 下葬、祭祀、扫墓 |

## 6. 分类择日规则

### 婚嫁择日

**宜：**
- 天德、月德、三合、六合日
- 成日、开日、定日
- 黄道吉日

**忌：**
- 月破、岁破、四离四绝
- 建日、破日、闭日
- 冲男女双方生肖

**首选：** 春秋两季，避开三伏天和数九天

### 开业择日

**宜：**
- 成日、开日、满日
- 天德、月德、天赦日
- 财日、禄日

**忌：**
- 破日、闭日、建日
- 岁破、月破
- 冲店主生肖

**首选：** 春季（生发）或秋季（收获）

### 动土择日

**宜：**
- 执日、成日、开日
- 天德、月德、天赦
- 宜动土的黄道日

**忌：**
- 破日、危日、建日
- 三煞方、太岁方
- 冲屋主生肖

**首选：** 避开太岁方位和五黄方位

### 出行择日

**宜：**
- 开日、成日、除日
- 驿马日、天德日
- 黄道吉日

**忌：**
- 破日、闭日、危日
- 岁破、月破
- 冲出行者生肖

**首选：** 避开五不归日和往亡日
