## 🔗 联动规则

### 1. 八字 + 紫微（命理互参）

**使用场景：** 深度性格分析、命运趋势推断

**联动逻辑：**
```
1. 先排八字，确定日主五行、格局、喜用神
2. 再排紫微，查看命宫主星、十二宫分布
3. 综合判断：
   - 八字日主 + 紫微命宫 = 核心性格
   - 八字财星 + 紫微财帛宫 = 财运分析
   - 八字夫妻星 + 紫微夫妻宫 = 婚姻分析
```

**示例断语：**
> "八字日主壬水，聪明灵活；紫微命宫天同，温和善良。两者相合，主其人外柔内刚，适应力强，人缘佳。"

---

### 2. 奇门 + 六爻（事项占卜）

**使用场景：** 具体事项决策、吉凶判断

**联动逻辑：**
```
1. 先起奇门，看时局大势、方位吉凶
2. 再起六爻，看具体事项成败
3. 综合判断：
   - 奇门吉门吉方 + 六爻吉卦 = 大吉，可行
   - 奇门凶门 + 六爻凶卦 = 大凶，宜止
   - 奇门吉 + 六爻凶 = 大势好但细节有问题，需谨慎
```

**示例断语：**
> "奇门开门落震宫，主事业有进展；六爻得『火天大有』，诸事顺遂。两者相合，此事可成，宜积极进取。"

---

### 3. 八字 + 风水（环境布局）

**使用场景：** 居家/办公风水调整

**联动逻辑：**
```
1. 先排八字，确定喜用神五行
2. 再排风水，查看宅向、九星落宫
3. 综合调整：
   - 喜木：东方/东南方宜摆放绿植
   - 喜火：南方宜用红色装饰
   - 喜土：中央/西南/东北宜用黄色
   - 喜金：西方/西北方宜用金属装饰
   - 喜水：北方宜用水景/黑色
```

**示例断语：**
> "八字喜用神为木火，宅向坐北朝南。建议：东方摆放绿植助木气，南方用红色装饰助火气，可旺事业财运。"

---

### 4. 奇门 + 择日（择吉时方）

**使用场景：** 开业、嫁娶、出行、动土

**联动逻辑：**
```
1. 先查择日，选黄道吉日
2. 再排奇门，选吉时吉方
3. 综合选择：
   - 黄道吉日 + 奇门吉时 = 最佳时机
   - 避开凶日凶时
```

**示例断语：**
> "4 月 14 日值『天德合』，黄道吉日；奇门开门落午宫，午时（11:00-13:00）最吉。建议此日午时开业，面向南方，大吉。"

---

### 5. 太乙 + 奇门（大势判断）

**使用场景：** 国运、经济、天灾等宏观预测

**联动逻辑：**
```
1. 先排太乙，看大势走向
2. 再排奇门，看具体时空
3. 综合判断：
   - 太乙主算 + 奇门值符 = 大趋势
   - 太乙客算 + 奇门值使 = 具体应期
```

## 🎯 统一断语风格规范 v3.1.0

### 断语原则

1. **言简意赅** —— 简明扼要，不拖沓
2. **古风韵味** —— 略带文言，但不晦涩
3. **切中要害** —— 直指核心，不绕弯
4. **趋吉避凶** —— 给出可行建议

### 断语模板

```
【总论】
一句话概括整体运势/命格特点

【分项】
• 【事业】...
• 【财运】...
• 【婚姻】...
• 【健康】...

【建议】
1. 宜...
2. 忌...
3. 趋吉方位/颜色/时机...

【卦理总结】
引用一句经文或卦辞，点明核心道理
```

### 示例

```
【总论】
此命金水相生，聪明智慧，事业有成之象。

【分项】
• 【事业】官星得地，利公职或管理工作
• 【财运】财库丰盈，中年后财运更佳
• 【婚姻】夫妻宫稳，宜晚婚
• 【健康】注意肾水系统保养

【建议】
1. 宜北方发展，从事水相关行业
2. 忌南方火地，避免急躁决策
3. 幸运色：黑、蓝；幸运数：1、6

【卦理总结】
"金水相生，智慧通达。顺势而为，自有成就。"
```
